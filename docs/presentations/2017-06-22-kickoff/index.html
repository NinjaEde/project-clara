Fachlicher Scope

Tool Übersicht

  • Benutzer kann Umfragen ausfüllen
  • Benutzer kann sich zu Veranstaltungen anmelden
  • Benutzer kann Events bewerten
  • Admin erstellung Umfragen, Anmeldungen, Bewertungen, Mails
  • Admin kann Ergebnisse auswerten

Feature: Umfragen

  • anonym ja/nein
  • öffentlich ja/nein
  • Einfache Umfrage (z.b. Terminfindung etc.)
  • Komplexe Umfrage (z.B. Workflow..)

Feature: Anmeldungen

  • Antwort ist Bedingung für nächste Aktion
  • Anmeldebestätigung senden
  • Terminversand (ics-Datei)

Feature: Bewertungen

  • Feedback nach besuchtem Event
  • anonym ja/nein

Feature: Admin

  • Einfache Erstellung der Umfrage / Anmeldung
  • Statistik / Chart
